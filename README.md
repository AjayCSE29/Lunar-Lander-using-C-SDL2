# Lunar-Lander-using-C-SDL2
Fan remake of the game Lunar Lander released in 1979 with only C++  

<img width="3264" height="1920" alt="re_menu" src="https://github.com/user-attachments/assets/47e514fe-fb8f-48a4-aafa-0c4ca00140e3" />
    
Initially planned to be made with the *Godot* engine but switched later to be made entirely with only C++  
The game consists a total of four scenes,  
                          1.  MENU
                          2.  LOADING
                          3.  GAME
                          4.  WIN

  <img width="1024" height="1536" alt="file_00000000f6c47208b921184e9eccd6f0" src="https://github.com/user-attachments/assets/c0e3c254-5036-4697-9ba5-c3657659b9aa" />  
This is the overall flow of the game logic.  

The Game is made with mostly SDL2 libraries like SDL_image, SDL_mixer and SDL_ttf.  
This is a windowed game with dimensions 1088x640
